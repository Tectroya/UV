## ⚡ ¿Por qué UV?

- **10x más rápido** que pip en la instalación de paquetes
- **Gestión de versiones de Python** integrada (sin pyenv)
- **Lock files automáticos** para reproducibilidad garantizada
- **Una sola herramienta** para reemplazar pip, virtualenv, pyenv y pipx
- **CI/CD ultrarrápido** con caché de dependencias
